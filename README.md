# three 项目开发

github:https://github.com/nullno/three-cli

demo:https://source.nullno.com/three-demo/


# 使用

```
 开发 npm run dev
 打包 npm run build 

```

# 文件说明
src/lib.js 自定义three文件，跟据需要引入模型loaders依赖或其他组件

src/index.js 入口文件

src/model 模型资源文件夹， 最好是通过cdn引入

src/environment 环境贴图hdr
